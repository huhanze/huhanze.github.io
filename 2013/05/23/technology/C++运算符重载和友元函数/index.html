1.1 概念

       运算符重载是一种形式的C++多态。运算符重载将重载的概念扩展到运算符上，允许C++运算符多种含义。实际上很多C++运算符已经被重载，如：将*运算符用于地址，将得到存储在这个地址中的值；但将它用于两个数字时，得到的将是它们的乘积。

1.2 运算符重载注意事项

多数C++运算符都可以重载，重载的运算符(有些例外的情况)不必是成员函数,但必须有一个操作数是用户自定义的类型。

  1. 重载后的运算符必须至少有一个是用户自定义的类型，防止用户为标准类型重载运算符。
  2. 使用运算符重载不能违反运算符原来的句法规则，不能修改运算符的优先级。
  3. 不能创建新运算符。

  4. 不能重载下列运算符。

    • sizeof: sizeof运算符
    • .:成员运算符。
    • .*:成员指针运算符。
    • ::：作用域解析运算符。
    • ?:：条件运算符。
    • typeid: 一个RTTI运算符。
    • const_cast:强制类型转换符。
    • dynamic_cast:强制类型转换符。
    • reinterpret_cast:强制类型转换符。
    • static_cast:强制类型转换符。

  5. 大多数运算符都能通过成员或非成员函数进行重载，但下列运算符只能通过成员函数进行重载。

    • =：赋值运算符。
    • ():函数调用运算符。
    • []:下标运算符。
    • ->:通过指针访问类成员的运算符。
