1. 圈子小喇叭

需求：圈子页面增加小喇嘛入口，记录当前圈子用户送礼情况。

分析：

: 页面可采用UITabView，自定义UITableViewCell;由于每个Cell的控件保持不变，送礼区根据设备动态调整宽度，这里采用AutoLayout来布局子控件较为方便。

准备工作：

创建控制器 - BaseTableViewController 继承自BaseTableViewController
创建数据模型 - QLCircleTrumpetModel, 以及数据管理模型 - QLCircleTrumpetModelManager
自定义UITableViewCell - QLTrumpetTableViewCell

主要实现思路 ：利用 QLCircleTrumpetModelManager模型对网络请求部分进一步封装，尽量避免控制器与网络请求API直接交互，采用将数据在集中处理，大大减少了控制器层的代码，使控制器看起来更加清楚明了，降低代码间的耦合性。

主要代码实现 : QLCircleTrumpetModel.h



QLCircleTrumpetModel.m



通过一个类方法将网络请求部分封装起来，这样减少大量控制器中的代码，使控制器看起来更加简洁。

### 2. 圈子贡献榜
需求：修改公告，增加公告详情，增加贡献榜，进入圈子模块显示贡献榜排名前7位的用户头像，点击进入贡献榜排行榜；

公告页面右上角增加关闭按钮，每一次进入圈子页面显示公告页，用户点击关闭按钮可关闭公告页面，点击公告页面进入公告详情页。
贡献榜分为周榜和总榜样式同富豪榜。

分析：
>: 排行榜样式分为4中，前3名头像含描边，大小各不同，其余均为普通样式，则在数据模型中定义一个类型枚举，通过枚举常量来控制应显示的样式。



同样，这里定义了处理数据的模型QLRankListModel，网络请求管理模型QLRankListModelManager来处理相应的数据。
